# Grade (gradescope without scope)

Experimental project. Stabilities not gauranteed.

## Dependencies

- python 3.8+
- node 14+

## Run

```sh
cd electron
npm i 
# if you have yarn installed, you should use `yarn install` instead
cd ..
python3 -m pip install -r requirements.txt 
python3 main.py
```
